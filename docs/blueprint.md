# **App Name**: CourseMaster

## Core Features:

- Admin Dashboard: Provides an overview of key metrics and administrative functions, displaying general statistics of the entire platform
- Student Management: Allows admins to create, edit, delete, and list students. Includes detailed profiles and enrollment information.
- Teacher Management: Enables admins to create, edit, delete, and list teachers. Manages teacher assignments and course responsibilities.
- Course Enrollment: Allows students to enroll in available courses, with a user-friendly interface for browsing and selecting courses.
- Course Viewing: Enables students to view enrolled courses, access materials, and track progress.
- Personal Profiles: Allows users to manage personal details, update settings, and customize their experience. Teachers can view details of the assigned courses.
- Personalized Learning Schedule Generator: Uses generative AI to suggest optimal schedules based on enrolled courses and student availability. The model acts as a scheduling "tool" using the input information.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional and trustworthy feel.
- Background color: Light Gray (#F5F5F5) for a clean and modern look.
- Accent color: Yellow (#FFEB3B) to highlight important actions and information.
- Body and headline font: 'PT Sans' for a modern and slightly warm feel that suits both headlines and body text.
- Use consistent, clear icons from Material Design to represent different sections and actions.
- Employ a modular layout with separate layouts for Admin, Teacher, and Student roles to maintain distinct user experiences.
- Use subtle animations for transitions and interactions to enhance user engagement.
